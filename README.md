# Namma Kathey 📖

A Gen AI-powered storytelling app built with Kotlin that generates creative and engaging stories using artificial intelligence.

## 🧩 Problem it Solves

Many people — especially children and students — struggle to find personalized, engaging stories in their preferred language or genre. Namma Kathey uses Generative AI to create unique, on-demand stories tailored to the user's input, making storytelling accessible and fun for everyone.

## 👥 Who It's For

- Children who love bedtime stories
- Students who want to explore creative writing
- Anyone who enjoys AI-generated storytelling experiences

---

## ✨ Features

- 🤖 **AI Story Generation** — Generate unique stories instantly using Gen AI
- 📝 **Custom Prompts** — Enter your own story idea or character and let AI build around it
- 📚 **Multiple Genres** — Choose from adventure, fantasy, moral stories, and more
- 🌐 **Multilingual Support** — Stories can be generated in regional languages including Kannada
- 📱 **Clean Android UI** — Simple and intuitive interface built with Kotlin

---

## 🚀 Setup & Installation

### Prerequisites
- Android Studio installed
- Android device or emulator (API level 21+)
- Gemini API Key

### Steps

1. Clone the repository — `git clone https://github.com/rakshitharakshitha22759-wq/Namma-Kathey.git`

2. Open in Android Studio — Launch Android Studio, click `File > Open` and select the `NammaKathey` folder

3. Run the App — Connect an Android device or emulator, then click the **Run** button or press `Shift + F10`
